# ModularEvo Demo System

A demonstration system for modular model evolution, featuring model merging  and adaptive routing algorithms.

## Project Structure

- `algorithm/` - Core algorithms
- `backend/` - FastAPI backend server
- `frontend/` - Vue.js frontend application
- `data/` - Datasets and model files

# Start the system
start.sh

# Stop the system
stop.sh
